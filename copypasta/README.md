# 模板仓库

一个算法模板应该涵盖下面几点：
- 对该算法的基本介绍（核心思想、复杂度等）
- 参考链接或书籍章节（讲的比较好的资料）
- 模板代码（可以包含一些注释）
- 使用模板的注意事项
- 模板补充内容（常见题型中的额外代码、建模技巧等）
- 相关题目（模板题、经典题、建模转换题等）

注：为方便在比赛中使用，一部分函数写成了 lambda 的形式。
